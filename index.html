Emmetについて

EmmetとはHTMLやCSSなどを記述する際に単純な形式の記述を展開することによって素早く編集することができる、ソースコードエディタ用のプラグインです。
また、エディタによっては初めからEmmetが組み込まれているものもあります。
Emmet公式サイト

Emmetの具体的な例を２つ紹介します。

    HTML編集モードとは：

    ソースコードエディタ（これから挙げるVSCodeやAtomなど）では、
    新規書類を開いたときのモードがプレーンテキスト編集モードになっています。
    編集画面の右下に Plain Text とありますので、そこをクリックして、
    検索欄に HTML と入力してEnterキーを押します。
    HTML編集モードにすることでエディタが入力されたテキストをHTMLと認識します。

  1. エディタのHTML編集モードで次のコードを記述してEnter（エディタによってはTab）キーを押すと、

    section>h2+p>a

    次のHTMLに展開されます。

    <section>
  <h2></h2>
  <p><a href=""></a></p>
</section>

  2. エディタのHTML編集モードで次のコードを記述してEnter（エディタによってはTab）キーを押すと、

    !

    次のHTMLに展開されます。

    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>

</body>
</html>

ソースコードエディタについて

ソースコードエディタとは、プログラマがプログラムのソースコードを編集することを目的として設計されたテキストエディタプログラムです。

Emmetは様々なソースコードエディタにインストール出来るプラグインが用意されています。
Emmetプラグインのダウンロードページ
Visual Studio Code（通称VSCode）には標準でEmmetが組み込まれています。
ここからはVSCodeを使用して説明を進めていきますが、
VSCodeは人気のあるソースコードエディタの中で使い心地がよく筆者が愛用している、
というだけなので素晴らしいEmmet対応のソースコードエディタは他にもあります。
現在人気のあるEmmet対応のソースコードエディタの中で無料のものを３つ挙げます。
３つともmacOS、Windows、Linuxでコーディングすることが出来ます。

  • Visual Studio Code

    • Microsoftが開発したソースコードエディタです。

  • Atom

    • GitHubが開発したオープンソースのテキストエディタです。
    ダウンロード


  • Brackets

    • Adobe Systemsが開発するオープンソースのソースコードエディタです。
    ダウンロード

Visual Studio CodeのインストールとEmmetの設定

  1. インストール

    2. 次のリンク先からあなたがお使いのOSにあったファイルをダウンロードすることが出来ます。
    Visual Studio Code

    macOSの場合は、ダウンロードしたファイルをダブルクリックして開き、Visual Studio Codeをアプリケーションフォルダへドラッグ＆ドロップします。

    Windowsの場合は、Installerをダウンロードすることをおすすめします。
    使用しているパソコンがあなたが占有しているものなら、System Installerを、複数の人で共有しているのなら、User Installerをダウンロードします。
    ダウンロードしたInstallerを起動し、指示に従ってインストールします。

  2. Emmetのlang設定

    3. Emmetの具体的な例の2.で紹介しましたコードの2行目が、

    <html lang="en">
    になっています。
    これは、HTMLの言語設定を英語にする、ということなのですが、
    日本語用のHTMLを記述する場合、この部分を毎回、
    <html lang="ja">
    に手動で書き換える必要があります。
    ここでは、VSCodeの設定にEmmetの変数を指定することで、自動で
    <html lang="ja">
    にする方法を紹介します。

    （lang=Languageの略、en=Englishの略、ja=Japaneseの略）

    VSCodeを起動してキーボードからCtrl+,を入力します。
    検索欄に variables と入力します。
    Edit settings.jsonをクリックしてJSONファイルを開きます。
    ファイルに何も書かれていない場合は次のコードをコピー＆ペーストしてファイルを保存します。
    {  // Emmetの言語変数を日本語に
  "emmet.variables": {
    "Lang": "ja"
  }
}

    ファイルにすでに何か書いてある場合は、最初の{の後ろに次のコードをコピー＆ペーストしてファイルを保存します。
    // Emmetの言語変数を日本語に
  "emmet.variables": {
  "Lang": "ja"
},

Live Sass Compilerのインストール
